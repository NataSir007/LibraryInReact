# Bank Logos

This folder should contain bank logos for the authentication UI.

## Required Files

The following logo files are needed:
- `varmennekortti.svg` 
- `mobiilivarmenne.svg`
- `osuuspankki.svg` 
- `nordea.svg` 
- `danske.svg` 
- `alandsbanken.svg` 
- `spankki.svg`
- `aktia.svg`
- `pop.svg`
- `saastopankki.svg` 
- `oma.svg` 
- `hightrust.svg`

## How to Add Logos

1. Download official logos from each bank's brand/press page
2. Save them in client\public\assets\logos folder with the filenames listed above


**Note:** These files are not included in the repository due to copyright restrictions. You must obtain them separately.
